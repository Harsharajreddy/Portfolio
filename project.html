COVID-19 Detection from CT scans using RESNET50 & XCEPTION CNN Models

COVID-19 detection using Convolutional Neural Networks (CNNs) while addressing class imbalance issues. I utilize RESNET50 and XCEPTION architectures, integrating class-weighted approaches to rebalance COVID and non-COVID datasets. By tackling class imbalance, my methodology enhances model accuracy. Through rigorous experimentation, I demonstrate the efficacy of my approach in improving COVID-19 detection accuracy and mitigating biases inherent in imbalanced datasets. This project underscores my commitment to leveraging advanced CNN techniques for effective pandemic response.
